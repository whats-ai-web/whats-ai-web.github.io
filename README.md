# WhatsApp 翻译助手静态展示页

## 本地预览

```bash
cd official
python3 -m http.server 8080
```

浏览器访问：`http://localhost:8080`

## 资源说明

页面使用 `official/assets` 中的静态图片，不依赖后端接口。
`qrcode.png` 语义为“二维码登录演示截图”，不是下载二维码。
`wechat_qrcode.png` 语义为“微信添加好友联系方式二维码”。
